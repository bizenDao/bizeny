# BizenNFT with Token Bound Account

千年続く伝統文化を支えるブロックチェーン技術

## NFTの価値について

### NFTの本来の価値
- 95%のNFTが無価値と言われるのは、目的が投機だから
- BizenDAOの目的は陶器そのもの
- ブロックチェーンに刻む情報こそがNFTの本来の価値
- 美術品の歴史＝骨董品の価値の本質
- 次世代の人類に残すべき文化の歴史こそブロックチェーンに刻むに相応しい

### ビジョン
- NFTに本来の価値を乗せて、遠い未来の人類に現代の歴史を伝えたい

## データストレージ — Arweave

### なぜArweaveか
- 陶器は人の一生よりずっと長く続く — 遠い未来を見据えたデータストレージが必要
- Arweaveは革新的な分散型ストレージ
- ブロックチェーン技術でデータの永続性とセキュリティを最大限に強化
- 情報を改ざんから保護し、中央権威を排除
- 最低200年という超長期でデータの確実な保存を実現
- 一度の支払いで永久的なデータ保存（継続的な料金負担なし）

### 意義
- 200年後の人類のために、web3の先端技術はその先の未来を見つめている

## Token Bound Account (TBA) — ERC-6551

### TBAとは
- NFTに紐づいたコントラクトアカウント
- つまり、陶器が個々に独立し自らのウォレットを持つということ

### 活用
- 人生に華やかなイベントがあるように、愛器にも数多くのイベントがある
- ウォレットにPOAPが貯まるように、愛器のウォレットにも陶器が経験したイベントのPOAPを貯められる
- 共に過ごした経歴をブロックチェーンに刻むことで、愛器の歴史は大きく成長する

### 意義
- TBAで陶器の新たな歴史が始まる

## Soul Bound Token (SBT) — ERC-5192

### SBTとは
- 譲渡不可能なNFT（一度受信したらウォレットの外に移せない）
- Web3時代のアイデンティティの構成要素
- ユースケース: 賃貸契約、無担保ローン、大学の学位付与、DAO、イベントチケット等

### 備前焼でのSBT活用例
TBAを持つNFTに紐づいた備前焼は魂(SOUL)を持つことになる。

**譲渡不可な記録（TBAに格納）:**
- 鑑定書
- 作家認定
- 受賞歴
- 修復歴
- 出展記録
- 贈答記録

### 陶器のアイデンティティ
- TBAを持つNFTに紐づいた陶器はSBTによって人と同様のアイデンティティを持つ
- そのアイデンティティは陶器の経歴を証明し、人と同様の社会的信用を得る

### 人間と陶器の対等な社会的信用
**人間の社会的信用:** 首席卒業、資格・免許、博士号、受賞歴、発明・特許、職務経歴、ボランティア、勲章など

**陶器の社会的信用:** 鑑定書、保証書、出展歴、受賞歴、修復詳細、贈答記録、作家が人間国宝、歴史的人物所有、思い出の記録など
